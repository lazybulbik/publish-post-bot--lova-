Настройка бота:

	1. Распакуйте архив "project.zip"

	2. Получение токена Telegram:
		·Откройте Telegram и найдите бота под названием "@BotFather".
		·Начните диалог с "@BotFather", нажав кнопку "Start".
		·Отправьте команду "/newbot", чтобы создать нового бота.
		·Следуйте инструкциям "@BotFather", чтобы выбрать имя и получить токен для вашего бота.
		·Скопируйте полученный токен, он будет использоваться для настройки вашего бота
		
	3. Получение id администратора.
		·Найдите бота https://t.me/getmyid_bot
		·Начните диалог с ним, нажав кнопку "Start"
		·Скопируйте цифры напротив "Your user ID:"

	3. Редактирование файла config.py:
		·Откройте файл "config.py" в любом текстовом редакторе (можно в блокноте).
		·Найдите переменную "TELEGRAM_TOKEN".
		·Замените значение переменной TELEGRAM_TOKEN" на ваш полученный токен Telegram: TELEGRAM_TOKEN = 'ваш_токен'.
		·Замените значение переменной ADMIN_ID на id администратора
		·Сохраните и закройте файл
	
	4. Создание нового архива:
		·Выделите все файлы, распакованные ранее.
		·Создайте архив с любым названием.

Размещение на хостинге (Инструкция описывает размещение на хостинге "deploy-f"):
	1. Перейдите на сайт "https://v2.d-f.pw/"
	2. Нажмите на кнопку "Запустить приложение"
	3. Зарегестрируйтесь
	4. Слева перейдите в раздел "Баланс"
	5. В разделе "Тарифный план" выберите "Бронза" и пополните баланс.
	6. Слева перейдите а раздел "Добавить"
	7. Выберите тип вашего приложения "Python"
	8. Загрузите архив (Пункт 4 раздел "Настройка бота") на сайт
	9. Нажмите на кнопку "Изменить параметры" и в "Команда для запуска" выберите "python main.py" и нажмите "Далее"
	Произойдет сборка образа.
	10. Все готово! Нажмите на кнопку "Запустить" и бот заработает
